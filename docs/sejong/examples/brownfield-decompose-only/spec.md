# Uigwe Wrapper Surface Spec

## Goal

Define the next wrapper-facing surface for Uigwe so future tooling can invoke the protocol through a stable input and output contract without changing the existing protocol artifacts.

## In Scope

- wrapper invocation contract
- wrapper output and handoff description
- docs integration with the existing Uigwe package

## Out Of Scope

- implementing the wrapper itself
- changing packet or goal-tree formats
- redesigning Uigwe's planning lifecycle

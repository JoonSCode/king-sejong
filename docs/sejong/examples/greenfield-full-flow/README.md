# Greenfield Full Flow Example

This example shows Uigwe entering from the very beginning of the planning lifecycle.

## Entry Mode

`full`

## Scenario

Create a greenfield MVP for async interview practice, where a user can run a short structured interview and receive rubric-based feedback.

## Artifact Order

1. `intent.packet.json`
2. `design.packet.json`
3. `spec.md`
4. `rationale.md`
5. `plan.packet.json`
6. `goal-tree.json`

## What This Example Demonstrates

- why `greenfield` gives more room for alternatives early on
- how a selected approach and retained alternatives appear in the design stage
- how decomposition turns that design into Seungjeongwon handoff leaves

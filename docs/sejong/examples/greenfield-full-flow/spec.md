# Async Interview Practice MVP Spec

## Goal

Launch a browser-based MVP that lets a user start a short async interview, answer a structured set of prompts, and receive rubric-based feedback with a saved summary.

## In Scope

- role-based interview setup
- question progression for one interview session
- answer capture
- rubric-based feedback summary
- saved session history

## Out Of Scope

- live voice interviewing
- marketplace or coach matching
- enterprise analytics

# Uigwe Validation Runs

Generated benchmark scorecards can live here.

The deterministic instruction-surface benchmark writes:

- `uigwe-instruction-surface.scorecard.json`
- `uigwe-instruction-surface.scorecard.md`

The deterministic Sejong surface benchmark writes:

- `sejong-surface.scorecard.json`
- `sejong-surface.scorecard.md`

For manual planning benchmark runs, use a dated subdirectory or a system-under-test name such as:

- `baseline-lightweight/`
- `baseline-structured/`
- `uigwe/`
- `sejong-uigwe/`
- `sejong-surface/`
